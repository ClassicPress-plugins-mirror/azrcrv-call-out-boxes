== Description ==

Place configurable call-out box in posts, pages or other post types.

Integrates with the Icons plugin from azurecurve for call-out box icons.

The following parameters can be used with the call-out-boxes shortcode:
* icon
* heading
* width
* margin
* padding
* border
* border-radius
* color
* background-color

== Installation ==
To install the Call-out Boxes plugin:
* Download the plugin from <a href='https://github.com/azurecurve/azrcrv-call-out-boxes/'>GitHub</a>.
* Upload the entire zip file using the Plugins upload function in your ClassicPress admin panel.
* Activate the plugin.

= Can I translate this plugin? =
* Yes, the .pot fie is in the plugin's languages folder and can also be downloaded from the plugin page on https://development.azurecurve.co.uk/; if you do translate this plugin please sent the .po and .mo files to translations@azurecurve.co.uk for inclusion in the next version (full credit will be given).
